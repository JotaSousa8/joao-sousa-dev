Cases · Defined.ai

.NET 10 migration & GDPR-ready platform

Senior Software Engineer · 04.2024 — Present

30+

Microservices migrated to .NET 10

40+

Services in the ecosystem

GDPR

Privacy solution designed

Problem

A large microservices estate needed to move off legacy .NET versions, stay operable at scale, and meet stronger privacy expectations for sensitive user data — without freezing delivery.

Approach

  • Migrated 30+ microservices from legacy .NET versions to .NET 10.
  • Designed architecture across a complex ecosystem of 40+ microservices, including service boundaries and cloud-native deployment on Azure.
  • Designed a GDPR solution covering data handling and compliance needs across the platform.
  • Supported infrastructure evolution (Azure Classic → ARM) with Docker, Kubernetes, and resilient messaging via RabbitMQ.
  • Mentored junior engineers and aligned technical work with business priorities.

Outcome

A modernized service fleet on .NET 10, clearer architecture across 40+ services, and a GDPR-ready approach to sensitive data — built to scale with the product rather than fight it.

.NET 10 · WebAPI · SQL Server · Docker · RabbitMQ · Azure · Kubernetes · EF · Dapper

Cases · Farfetch

Recommendations at ~50ms

Software Engineer .NET · 09.2020 — 03.2024

~50ms

Recommendation response time

High

Throughput in production

ML

Close DS / ML collaboration

Problem

Recommendation systems had to stay fast and reliable under heavy traffic, while integrating evolving ML models and a heterogeneous data landscape.

Approach

  • Built and maintained mission-critical .NET recommendation services targeting ~50ms response times at high throughput.
  • Collaborated tightly with Data Science and Machine Learning on architecture and model integration.
  • Managed polyglot persistence — MongoDB, SQL, Cassandra, Redis — for the right store per access pattern.
  • Used Kafka, Docker, and Kubernetes for distributed, cloud-native delivery; brought Python into production data workflows where it fit.

Outcome

A production recommendation path delivering around 50ms responses under high throughput — stable enough to absorb ML-driven change without sacrificing the shopping experience.

.NET · WebAPI · Kafka · MongoDB · Cassandra · Redis · Kubernetes · Python · Jenkins
